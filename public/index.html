How it works...

This boiler plate comes with one pre-defined x-html tag usable via <x-text-html/> and having several attributes and behaviors pre-defined.
You can view the source for this tag within YourProject/src/x-tags/TextHtml

The lifecycle of the tag goes something like this…

  1. Upon instantiation, the required 'src' attribute is used to load JSON from a server.
  2. Next, the optional 'property' attribute is used to target a property within the JSON.
  3. Note, that if the optional 'property' target is not set then all the data is displayed.
